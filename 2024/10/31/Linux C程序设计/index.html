Linux C程序设计

sudo su进入管理员

pwd 显示当前目录

cd .. 到上一级目录

cd ++可以到相应的目录下

ls命令

ls 查看当前目录下有什么文件

ls -l 和ls -lh 查看当前目录里面文件的具体信息

ls -a 查看当前目录的隐藏文件

cat命令

cat test.c查看这个文件

cat > test2.c 创建文件

cat test.c test.c > test3.c 将两个文件合并并创建出一个新文件

cat >> test3.c 在已经创建的文件中添加内容

more命令

more test3.c查看当前文件

more -3 test3.c 从文件的第三行开始查询

more +5 test3.c 跳过该文件的前三个文件

head命令

head test3.c 默认查看前10行

head -n 11 test3.c 修改为查看前11行

head -v test3.c 在内容头输出文件头

tail命令

tail test3.c从文件内容的末尾开始输出,默认输出10行

tail -n 3 test3.c修改输出行数为3

man tail查看命令

grep命令

查询文件内容

-i 忽略大小写

-v 只显示不匹配的行

-n 显示匹配行的行号
